Introduction
-----------

QWIM Dashboard is an interactive web application for time series visualization
and analysis. Built with Shiny for Python, it provides an intuitive interface
for exploring and analyzing financial time series data.

Installation
-----------

.. code-block:: bash

   pip install qwim-dashboard

Getting Started
--------------

To start the dashboard locally:

.. code-block:: bash

   python -m src.dashboard.main_App
